## Table Names and Primary Keys:

1. Hotels: separate id in number type for each hotel
2. Customers: separate customer id in number type (cause names or other info might be similar for different customers)
3. Meal_plans: separate id in number type for each meal plan
4. Room_types: separate id in number type for each room type
5. Bookings: separate id in num type (same reason as in customers)
6. Agents: agent id (it compounds only that information)
7. Companies: company id (it compounds only that information)
